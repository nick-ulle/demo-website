An Example of D3

Created by: Nick Ulle

This page shows an example of a D3 plot.

The code for this page is in this GitHub repository. The JavaScript code is in d3_example.js and the CSS code is in d3_example.css.

In order to view the D3 plot on this page on your local machine, the page must be served by an HTTP server. If you have Python 3 installed, you can launch a web server by running python -m http.server from the same directory as this HTML file. Then you can view the page at http://localhost:8000/.

The plot is based on this tutorial for an older version of D3.
